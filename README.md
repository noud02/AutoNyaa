# AutoNyaa

Automatic nyaa.si anime downloader.

## Installation

To install, run:

```bash
$ npm i -g yuwui/AutoNyaa
```

### Usage

```
autonyaa <anime[,anime,anime,...]>

Flags:
    --debug         Log warnings
    --user  <user>  Nyaa.si user to download from
    --res   <res>   Resolution to download
```
